### Brunch RSS Feed 생성
---

## 프로젝트 소개
브런치 웹사이트(https://brunch.co.kr/)에 업데이트되는 최신글들의 RSS Feed 를 생성해주는 서비스
## 프로젝트 영상

## 프로젝트 구성

- 사용 언어 및 프로그램 
	- Django
	- Docker
	- PostgreSQL
	- Python
	- RDS
	- S3
- 주요내용
	- 브런치 크롤링: headless selenium, Python Async 비동기 http 요청 
	- RSS Feed 생성: Django Feed Generator 를 커스터마이징하여 Feed(Keword, Writer) 생성
